# CLI HOTKEYS

-------

## Move around

- CTRL A / E    -     move - line home/end
- CTRL F / B    -     move - char for/backward
- ALT  F / B    -     move - word for/backward

-------

## Editing

- CTRL H        -     delete - char
- CTRL W        -     delete - word back
- CTRL K        -     clear - line -> end
- CTRL U        -     clear - line

-------

## Processes?

- CTRL C
- CTRL Z

-------

## Screen outputs

- CTRL S / Q    -     toggle screen output
- CTRL L        -     clear screen output

# CLI HOTKEYS

-------

## Move around

|  Commands  |          Actions         |
|:----------:|:------------------------:|
| Ctrl-A / E |   move - line home/end   |
| Ctrl-F / B | move - char for/backward |
| Alt -F / B | move - word for/backward |

-------

## Editing

| Commands |       Actions       |
|:--------:|:-------------------:|
|  Ctrl-H  |    delete - char    |
|  Ctrl-W  |  delete - word back |
|  Ctrl-K  | clear - line -> end |
|  Ctrl-U  |     clear - line    |

# YouTube

## Hotkeys

Besides the easy ones (for kids), these are more helpful hotkeys for advanced control:

|         Command         | Action |
|:-----------------------:|:------:|
|  Previous / Next video  |  P / N |
| Rewind / Forward 10secs |  j / l |
|       Play / Pause      |    k   |
|      Playback rate      |  < / > |
|      Caption's size     |  + / - |
|    Caption's opacity    |  o / w |
|    Keyboard shortcuts   |    ?   |

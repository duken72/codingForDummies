| :----------: | :--------------: |
| Ctrl+Shift+R |      \right      |
| Ctrl+Shift+L |      \left       |
| Ctrl+Shift+T |    \texttt{}     |
| Ctrl+Shift+A |    \textsf{}     |
| Ctrl+Shift+C |    \textsc{}     |
| Ctrl+Shift+U |       ^{}        |
| Ctrl+Shift+D |       \_{}       |
| Ctrl+Shift+I |      \item       |
| Ctrl+Shift+F | \dfrac{num}{den} |
| Ctrl+Shift+N | \begin{equation} |
| Ctrl+Shift+M |       $ $        |

# VIM

CLI commands:

```bash
alias vimg='less $(fd vimFD.bash ~)'
alias lessg='less $(fd lessFD.bash ~)'
```

-----

Check [vimFD.bash](vimFD.bash) and [lessFD.bash](lessFD.bash) for useful hotkeys.
